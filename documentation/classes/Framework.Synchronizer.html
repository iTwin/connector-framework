recordDocument

  • recordDocument(scope: string, sourceItem: SourceItem, kind?: string, knownUrn?: string): SynchronizationResults
    • Defined in Synchronizer.ts:99

    Insert or update a RepositoryLink element to represent the source document. Also inserts or updates an ExternalSourceAspect for provenance.

    throws

    [[IModelError]] if a RepositoryLink for this document already exists, but there is no matching ExternalSourceAspect.

    Parameters

    • scope: string

      The scope of the RepositoryLink's ExternalSourceAspect

    • sourceItem: SourceItem

      Identifies the document. The item's version property should the last modified time of the document. The checksum property is optional. It may be a cryptographic hash of the file's contents, or some other metric that is guaranteed to change when the document changes.

    • kind: string = "DocumentWithBeGuid"

      Optional. The document kind. Defaults to "DocumentWithBeGuid"

    • knownUrn: string = ""

      Optional. The URN of the master document. Defaults to "".

    Returns SynchronizationResults

updateIModel

  • updateIModel(results: SynchronizationResults, scope: string, sourceItem: SourceItem, kind: string, externalSourceElement?: ExternalSourceProps): IModelStatus
    • Defined in Synchronizer.ts:209

    Update the BIM with the results of converting an item to one or more DgnElements. If the item is new or changed, the conversion writes are written to the BIM and the associated ExternalSourceAspect is updated. If the item is known and unchanged, then the BIM is not updated. In either case, this function will record the id of the element as having been seen.

    beta

    Parameters

    • results: SynchronizationResults
    • scope: string

      Id of the scoping element

    • sourceItem: SourceItem

      Defines the source item

    • kind: string

      The kind of the source item

    • Optional externalSourceElement: ExternalSourceProps

      External source pointing to the Repository Link. Use getExternalSourceElement for this parameter

    Returns IModelStatus
